# NuxtJS

> 基于vue的一个服务端渲染框架 （nuxtjs服务端就是借助nodejs来进行页面组装）

### 优点

可以解决SEO问题

### 缺点

导致服务器压力变大



### 脚手架工具

```bash
npm i create-nuxt-app -g
```





默认监听3000端口，可以更改端口

```bash
"dev": "nuxt",

"dev": "nuxt port",
```



## 目录结构

### Components  组件目录

组件目录 components 用于组织应用的 Vue.js 组件。Nuxt.js 不会扩展增强该目录下 Vue.js 组件，即这些组件不会像页面组件那样有 asyncData 方法的特性。
